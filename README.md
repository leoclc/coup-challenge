# Coup Challenge

Webservice that calculates total number of engineers needed to maintain X number of scooters in berlin.  	  

## Getting Started

```
mvn install
mvn spring-boot:run

(Embedded Tomcat) configured on port 8080

```


## Running 

http://localhost:8080/  

JSON Example: 

```
{ scooters: [15, 10],
	C: 12,
	P: 5
}
```
